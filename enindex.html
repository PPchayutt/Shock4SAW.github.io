⊛ Main Function

The "Shock4SAW" Quiz Game Project is a quiz game developed using an Arduino board. The game displays questions on a display screen and provides options for players to answer by pressing buttons. When a correct answer is selected, a green LED lights up, a warning sound is heard, and the player advances to the next question. However, when an incorrect answer is selected, a red LED lights up, a warning sound is heard, and the player's finger is shocked through a shock circuit box. This is safe, as it uses a low DC voltage and uses a pre-programmed shock circuit. Players must hold their fingers against the contact points throughout the game. After answering all questions on the display screen, a score is displayed for the player's answers.

⊛ Objective

This project addresses the issue of lack of understanding of lessons, as misunderstanding can lead to poor test scores. The students who participate in this project benefit from this project, who are constantly stimulated and receive immediate feedback on correct or incorrect answers, which helps them retain the material better.
